[ 전문적인 주식 분석 인포그래픽 이미지 만들기 프롬프트 ]

① 기술적 분석 결과를 먼저 만들고 → ② 그 결과를 인포그래픽으로 시각화하는 2단계 프롬프트

-----------------------------------------------------------
1단계: 기술적 분석 리포트 생성 프롬프트
-----------------------------------------------------------

너는 주식 기술적 분석가다.

아래 종목 데이터를 기준으로 기술적 분석 리포트를 작성해줘.

[분석 대상]
- 종목명: 삼성전자
- 기준 차트: 일봉
- 기준일: 최신 거래일
- 데이터: OHLCV 기준

[분석 지표]
1. 추세
- EMA 5, 20, 60, 120
- 정배열/역배열 여부
- 골든크로스/데드크로스 여부
- 현재가가 주요 이동평균선 위/아래인지

2. 모멘텀
- RSI(14)
- MACD
- MACD Signal
- Histogram
- 과매수/과매도 여부
- 다이버전스 가능성

3. 변동성
- Bollinger Bands(20, 2)
- ATR(14)
- 현재가가 상단/중단/하단 밴드 중 어디에 있는지
- 스퀴즈/확장 여부

4. 수급/거래량
- 최근 거래량이 평균 거래량 대비 증가/감소했는지
- 가격 상승과 거래량 증가가 동반되는지
- OBV 흐름 해석

[출력 형식]
다음 구조로 정리해줘.

1. 상단 요약
- 현재가
- 고가
- 저가
- 등락률
- 거래량

2. 4축 기술적 분석 프레임워크
- 추세
- 모멘텀
- 변동성
- 수급

각 항목마다:
- 핵심 지표
- 현재 상태
- 해석
- 신뢰도 또는 주의점

3. 핵심 구조 분석
- 주요 저항 3개
- 주요 지지 3개
- 현재 위치
- 추세 전환 포인트

4. 시나리오 분석
- 상승 시나리오
- 하락 시나리오
- 각각의 조건과 목표가

5. 종합 판단
- 현재 국면
- 진입 신호 강도
- 핵심 관찰 포인트
- 리스크 포인트

6. 매매 전략
- 공격형
- 보수형
- 관망형

[주의사항]
- 확정적인 투자 조언처럼 쓰지 말 것
- “매수하라/매도하라” 대신 조건부 시나리오로 표현할 것
- 데이터가 불확실한 부분은 추정이라고 표시할 것
- 한국어로 작성할 것

-----------------------------------------------------------
2단계: 인포그래픽 이미지 생성 프롬프트
-----------------------------------------------------------

위의 기술적 분석 리포트를 바탕으로 전문적인 주식 분석 인포그래픽을 만들어줘.

[디자인 목표]
- 한국어 주식 기술적 분석 리포트 스타일
- 교육용/리포트용 인포그래픽
- 한눈에 핵심 판단이 보이도록 구성
- 너무 장식적이지 않고, 증권사 리포트처럼 신뢰감 있게
- 색상은 상승=빨강, 하락=파랑 또는 초록/빨강 계열을 명확히 구분
- 카드형 레이아웃
- 16:9 비율
- PPT 슬라이드에 바로 넣을 수 있는 구성

[상단 영역]
- 제목: 삼성전자 | 기술적 분석 리포트
- 기준: 일봉 차트
- 현재가, 고가, 저가, 등락률, 거래량을 숫자 카드로 표시
- 오른쪽에는 작은 일봉 차트 영역 배치
- 차트에는 이동평균선, 볼린저밴드, 최근 돌파 위치를 표현

[본문 1]
제목: [1] 4축 기술적 분석 프레임워크

4개 카드로 구성:
1. 추세 Trend
2. 모멘텀 Momentum
3. 변동성 Volatility
4. 수급 Volume & Flow

각 카드에는 핵심 bullet 4~5개와 최종 상태 라벨을 넣어줘.

[본문 2]
제목: [2] 핵심 구조 분석

왼쪽:
- 주요 저항 1
- 주요 저항 2

가운데:
- 주요 지지 1
- 주요 지지 2
- 주요 지지 3

오른쪽:
- 현재 위치
- 추세 전환 포인트

[본문 3]
제목: [3] 시나리오 분석

왼쪽 카드:
- 상승 시나리오
- 조건
- 1차 목표
- 2차 목표
- 황소 아이콘 또는 상승 그래프 아이콘

오른쪽 카드:
- 하락 시나리오
- 조건
- 1차 하락 목표
- 2차 하락 목표
- 곰 아이콘 또는 하락 그래프 아이콘

[본문 4]
제목: [4] 종합 판단

- 현재 국면
- 진입 신호 강도
- 핵심 관찰 포인트
- 리스크 포인트
- 별점 형태로 신호 강도 표현

[본문 5]
제목: [5] 매매 전략

3개 카드:
1. 공격형
2. 보수형
3. 관망형

각 카드에 조건과 대응 전략을 짧게 표시.

[하단 강조 문구]
“추세는 매우 좋지만, 지금은 ‘좋은 종목’보다 ‘좋은 자리’가 더 중요한 구간.”

[텍스트 스타일]
- 한국어가 깨지지 않게 선명하게 표시
- 숫자는 크게
- 핵심 단어는 굵게
- 과도한 문장보다 짧은 bullet 중심
- 전체적으로 깨끗하고 정돈된 금융 리포트 느낌
